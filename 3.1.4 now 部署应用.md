# now 部署应用

## install

```
npm i -g now


now login


now


------------------------------
---------now help-------------
------------------------------
 Examples:

 – Deploy the current directory

   $ now

 – Deploy a custom path

   $ now /usr/src/project

 – Deploy with environment variables

   $ now -e NODE_ENV=production -e SECRET=@mysql-secret

 – Show the usage information for the sub command `list`

   $ now help list
```
