# node 控制台打印表格

* word-table: https://github.com/hustcc/word-table

* use 

```
var WordTable = require('word-table');
var header = ['id', 'name', 'birthday'];
var body = [
  ['#1', '王小为', '1992-08-01', '备注：hustcc'], 
  ['#2', '小泥巴', '1992-09-20'],
  ['#3', '佚名', '保密']
];

// basic usage
var wt = new WordTable(header, body);
console.log(wt.string());

保存为 test.js
```
