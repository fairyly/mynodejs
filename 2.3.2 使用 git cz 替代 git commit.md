# 2.3.2 使用 git cz 替代 git commit

>看到 vue 源码中使用的
```
"scripts": {
  commit": "git-cz"
},
"devDependencies": {
  ...
  "cz-conventional-changelog": "^2.0.0",
},
"config": {
  "commitizen": {
    "path": "./node_modules/cz-conventional-changelog"
  }
}
```
## install

```
npm install --save-dev cz-conventional-changelog
```
