# 8.1.8 stf-从您的浏览器控制和管理Android设备

>

## Installation

```
npm install -g stf
```


## run

```
stf local


#If you want to access STF from other machines, you can add the --public-ip option for quick testing.

stf local --public-ip <your_internal_network_ip_here>
```
